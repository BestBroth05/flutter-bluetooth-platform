# flutter-bluetooth-platform

Personal portfolio project demonstrating a reusable Flutter Bluetooth Low Energy (BLE) architecture.

This repository is greenfield code. It is not affiliated with any commercial or medical product.

## Package identity

- Repository: `flutter-bluetooth-platform`
- Flutter package name: `bluetooth_platform`
- Application identifier: `dev.brayanolivares.bluetooth_platform`

## Current phase

Phase 1 focuses on architectural foundations and **simulator mode** (fake BLE transport). Real-device BLE via `flutter_blue_plus` is intentionally deferred.
